## 담당 모듈 
#### 학생 사물함 (등록, 자리이동, 기간연장, 삭제), 일반회원 홈 화면 및 관리자 대시보드 화면 프론트 및 백엔드 구현
#### Spring Quartz 스케줄러를 사용하여 사용종료 락커 자동삭제, 락커 사용종료 3일전 안내메일 전송 기능 구현
#### D3.js 라이브러리를 활용하여 관리자 페이지 데이터 시각화 구현
#### 트랜잭션 및 Select for Update 를 사용하여 동시성 문제 해결
#### 다양한 돌발 상황에 대응하는 예외 처리 구현
#### 락커 데이터를 삭제하지않고 모든 이력을 남기도록 설계하여 추후 통계에 활용할 수 있도록 함 (선호하는 자리 등)
#### DB모델링, CSS Flex Box, 자바스크립트 등 팀원들이 어려워하는 부분에 대해 자료를 정리하고 설명
#### 요구사항 분석부터 설계, 구현, 테스트, 유지보수에 이르기까지 소프트웨어 개발의 전 과정을 경험
#### 첫 팀 프로젝트임에도 최종 발표까지 잘 마무리 해준 팀원들에게 정말 고맙고 팀 협업의 효과와 중요성에 대해 깊이 이해하는 계기가 되었음
